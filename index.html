Nomos AI Pro REST API
Version 1.0 · Base URL: https://api.nomosai.ke/v1
● Live

The Nomos AI Pro API gives your systems direct access to Kenya's most powerful legal intelligence engine — contract analysis, document generation, case assessment, and legal research, all via a simple REST interface.

🔐 Authentication

All API requests must include your API key in the Authorization header:

Authorization: Bearer YOUR_API_KEY
📡 Endpoints
POST /v1/analyse Contract & Document Analysis
curl -X POST https://api.nomosai.ke/v1/analyse \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"document_text": "...", "analysis_type": "contract"}'
POST /v1/assess Case Strength Assessment
curl -X POST https://api.nomosai.ke/v1/assess \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"case_type": "tenant_rights", "description": "...", "county": "Nairobi"}'
POST /v1/generate Legal Document Generation
curl -X POST https://api.nomosai.ke/v1/generate \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"doc_type": "nda", "parties": [...], "jurisdiction": "Kenya"}'
GET /v1/precedents Kenya Law Precedent Search
curl https://api.nomosai.ke/v1/precedents?q=eviction&county=Nairobi&limit=10 \
  -H "Authorization: Bearer YOUR_API_KEY"
